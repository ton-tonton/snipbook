# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Purpose

This repository is a collection of book-style summaries generated from YouTube videos using the `youtube-summarizer` skill. Each video is summarized into a Markdown file named after the book or topic covered.

## Workflow

To create a new summary from a YouTube video:

1. Use the `youtube-summarizer` skill with the video URL.
2. Save the output as `[book-name].md` in the repository root (or appropriate subdirectory).
3. The language of the `.md` file must match the original language spoken in the video — do not translate.

## File Naming

- Use kebab-case for filenames: `the-lean-startup.md`, `atomic-habits.md`
- File names should reflect the book or topic title from the video content

## Content Language

The `.md` file content must be written in the same language as the original video. If the video is in Japanese, the summary is in Japanese. If in English, the summary is in English.
