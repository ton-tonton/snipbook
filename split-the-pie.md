## 📝 สรุปโดยรวม

หนังสือ *Split the Pie* โดย Barry Nalebuff นำเสนอแนวคิดการเจรจาต่อรองแบบใหม่ที่พลิกโฉมความเข้าใจเดิมๆ แทนที่จะมองการเจรจาว่าเป็นการแย่งชิงทรัพยากรจำกัด (Zero-Sum Game) ที่ต้องมีผู้ชนะและผู้แพ้ Nalebuff เสนอว่าการเจรจาที่ดีที่สุดคือการ **ขยายพาย** ก่อน แล้วค่อย **แบ่งพายอย่างเท่าเทียม** ซึ่งนำไปสู่ผลลัพธ์ที่ทุกฝ่ายได้ประโยชน์ร่วมกัน (Positive-Sum Game)

แก่นของหนังสือเล่มนี้คือการเปลี่ยนคำถามจาก "ฉันจะได้อะไร?" เป็น "เราจะสร้างมูลค่าร่วมกันได้อย่างไร?"

## 📖 เนื้อหาโดยละเอียด

### 1. ทำไมคนส่วนใหญ่เกลียดการต่อรอง?

คนส่วนใหญ่รู้สึกอึดอัดและกังวลเมื่อต้องเจรจาต่อรอง เพราะถูกปลูกฝังให้มองการเจรจาว่าเป็น **การสู้รบ** ที่มีคนแพ้และคนชนะ ความกลัวหลักๆ ได้แก่:

- **กลัวถูกเอาเปรียบ** — กลัวว่าอีกฝ่ายจะได้ส่วนแบ่งมากกว่า
- **กลัวความขัดแย้ง** — ไม่อยากทำให้ความสัมพันธ์เสียหาย
- **กลัวความล้มเหลว** — กลัวไม่ได้ข้อตกลงที่ต้องการ
- **ขาดความมั่นใจ** — ไม่รู้ว่าตัวเองมีอำนาจต่อรองแค่ไหน

Nalebuff ชี้ว่าความกลัวเหล่านี้เกิดจาก **กรอบความคิดที่ผิด** การมองการเจรจาเป็น Zero-Sum Game ทำให้ทั้งสองฝ่ายตั้งรับ ไม่กล้าเปิดเผยข้อมูล และสุดท้ายได้ผลลัพธ์ที่แย่กว่าที่ควรจะเป็น

### 2. กฎแห่งพาย: ทฤษฎีพิซซ่า Alice & Bob

นี่คือแก่นหลักของหนังสือ ลองนึกภาพสถานการณ์ง่ายๆ:

> **Alice และ Bob ต้องตกลงกันเรื่องการแบ่งพิซซ่า 1 ถาด ถ้าตกลงไม่ได้ภายในเวลาที่กำหนด ทั้งคู่ไม่ได้อะไรเลย**

**คำถามคือ: ควรแบ่งอย่างไรจึงจะยุติธรรม?**

**คำตอบของ Nalebuff:**

"พาย" ในที่นี้ไม่ใช่พิซซ่าทั้งถาด แต่คือ **มูลค่าที่เกิดขึ้นจากการตกลงร่วมกัน** (ZOPA — Zone of Possible Agreement) ซึ่งคำนวณได้จาก:

```
พาย = มูลค่าที่ได้จากดีล − มูลค่าที่แต่ละฝ่ายจะได้ถ้าไม่มีดีล (BATNA)
```

**ตัวอย่าง:**
- Alice มี BATNA (Best Alternative to a Negotiated Agreement) = 0 บาท
- Bob มี BATNA = 0 บาท
- ดีลที่เป็นไปได้สร้างมูลค่า = 100 บาท
- พาย = 100 − 0 − 0 = **100 บาท**
- การแบ่งที่ยุติธรรมที่สุด = **Alice 50 บาท, Bob 50 บาท**

**ทำไมต้องแบ่งเท่ากัน?**

เพราะทั้ง Alice และ Bob **ขาดไม่ได้เท่ากัน** ในการสร้างมูลค่านั้น ถ้า Alice ถอนตัว พายหายไป 100 บาท ถ้า Bob ถอนตัว พายก็หายไป 100 บาท เมื่ออำนาจในการสร้างพายเท่ากัน การแบ่งจึงควรเท่ากัน

**กรณีที่ BATNA ไม่เท่ากัน:**
```
มูลค่าดีล = 200 บาท
BATNA ของ Alice = 50 บาท
BATNA ของ Bob = 30 บาท
พาย = 200 − 50 − 30 = 120 บาท
การแบ่ง: Alice ได้ 50 + 60 = 110 บาท, Bob ได้ 30 + 60 = 90 บาท
```

BATNA ไม่ใช่อำนาจต่อรอง — มันเป็นแค่จุดตั้งต้น พาย (มูลค่าที่เพิ่มขึ้น) ต่างหากที่ควรแบ่งเท่ากัน

### 3. กรณีศึกษา: การขาย Honest Tea ให้ Coca-Cola (ขยายพายให้ใหญ่ขึ้น)

Barry Nalebuff ไม่ใช่แค่นักวิชาการ — เขาเป็นผู้ร่วมก่อตั้ง Honest Tea แบรนด์ชาออร์แกนิกที่ต่อมาขายให้ Coca-Cola ในราคาสูง

**บทเรียนสำคัญจากดีลนี้:**

**ก. ขยายพายก่อนแบ่งพาย**

ก่อนจะคุยเรื่องราคา Nalebuff โฟกัสที่การทำให้ดีลมีมูลค่าสูงสุดก่อน ซึ่งรวมถึง:
- **ช่องทางการจัดจำหน่าย** ของ Coca-Cola ที่ Honest Tea เข้าถึงไม่ได้เอง
- **ความสามารถในการขยายตลาดทั่วโลก**
- **มูลค่าแบรนด์ Honest Tea** ที่ Coca-Cola ต้องการเพื่อเข้าตลาด Health & Wellness

เมื่อทั้งสองฝ่ายเข้าใจว่าพายนั้นใหญ่แค่ไหน การแบ่งก็ง่ายขึ้นมาก

**ข. อย่าเปิดเผยราคาตั้งต้น**

ผู้ขายจำนวนมากพลาดตรงที่ตั้งราคาต่ำเกินไปเพราะ "กลัวตกลงไม่ได้" แต่จริงๆ แล้วการรู้ BATNA ของตัวเองและพายที่เกิดขึ้นช่วยให้รู้ว่าควรตั้งราคาที่ไหน

**ค. ความโปร่งใสสร้างความไว้วางใจ**

Nalebuff แชร์ข้อมูลเกี่ยวกับต้นทุนและกำไรกับ Coca-Cola อย่างเปิดเผย ซึ่งทำให้การเจรจาดำเนินไปได้เร็วขึ้นและสร้างความสัมพันธ์ระยะยาว

### 4. เมื่ออีกฝ่ายเล่นแรง: วิธีรับมือคนไม่มีเหตุผลและ Cybersquatter

ไม่ใช่ทุกการเจรจาจะราบรื่น Nalebuff นำเสนอกลยุทธ์รับมือสถานการณ์ยาก:

**ก. กรณี Cybersquatter**

เมื่อมีคนครอบครองโดเมนของ Honest Tea และต้องการค่าไถ่สูงเกินจริง:

- **วิเคราะห์พาย** — มูลค่าจริงของโดเมนสำหรับคุณคือเท่าไหร่? ต้นทุนของ Cybersquatter ในการถือครองต่อไปคือเท่าไหร่?
- **ตั้งคำถามเชิงหลักการ** — "ถ้าเราตกลงราคาตามหลักการ Split the Pie คุณคิดว่าเท่าไหร่ยุติธรรม?"
- **ชี้ให้เห็นทางเลือก** — แจ้งให้รู้ว่าคุณมีทางเลือกอื่น (BATNA) เช่น ใช้โดเมนชื่ออื่น หรือฟ้องร้องตาม UDRP

**ข. คนที่เล่นแรงหรือไม่มีเหตุผล**

- **อย่าตอบโต้ด้วยอารมณ์** — การขู่ขวัญกลับจะยิ่งทำให้สถานการณ์แย่ลง
- **กลับมาที่หลักการ** — "เรามาคุยกันในแง่ของมูลค่าที่สร้างร่วมกันดีกว่า"
- **ใช้เวลาเป็นเครื่องมือ** — บางครั้งการรอให้อีกฝ่ายเย็นลงช่วยได้
- **นำเสนอทางเลือก** — ให้อีกฝ่ายมีทางออกที่ "รักษาหน้า" ได้

**ค. เมื่อดูเหมือนอีกฝ่ายมีอำนาจมากกว่า**

หลายคนยอมแพ้ก่อนเริ่มเมื่อเห็นว่าอีกฝ่ายใหญ่กว่า แต่ Nalebuff เตือนว่าอำนาจที่แท้จริงวัดจาก **ความสำคัญในการสร้างพาย** ไม่ใช่ขนาดหรือความมั่งคั่ง

### 5. ความเข้าใจผิดเรื่อง "อำนาจ" ในการเจรจา

**ความเชื่อผิดๆ ที่พบบ่อย:**

| ความเชื่อเดิม | ความจริง |
|---|---|
| คนรวยกว่ามีอำนาจมากกว่า | อำนาจอยู่ที่ความสำคัญในดีล ไม่ใช่ความมั่งคั่ง |
| BATNA ดีกว่า = ได้เปรียบกว่า | BATNA เป็นแค่จุดตั้งต้น พายต่างหากที่ต้องแบ่ง |
| ต้องเปิดราคาสูงก่อนเสมอ | ขึ้นอยู่กับบริบท บางครั้งเปิดก่อนได้เปรียบ บางครั้งเสียเปรียบ |
| ข้อมูลคือความลับ | ความโปร่งใสมักสร้างมูลค่ามากกว่า |
| ยิ่งแข็งกร้าวยิ่งดี | ความร่วมมือมักสร้างผลลัพธ์ที่ดีกว่า |

**ตัวอย่างจากชีวิตจริง:**

บริษัท startup เล็กๆ อาจมีอำนาจต่อรองสูงมากกับบริษัทใหญ่ ถ้า startup นั้นมีเทคโนโลยีหรือทรัพยากรที่บริษัทใหญ่ต้องการจริงๆ — บริษัทใหญ่จะ "จำเป็น" ต้องมีดีลนี้มากกว่า ทำให้พายฝั่ง startup ใหญ่ขึ้น

**สูตรคำนวณอำนาจที่แท้จริง:**
```
อำนาจ = มูลค่าที่หายไปถ้าคุณถอนตัว / มูลค่าพายทั้งหมด
```

ถ้าการถอนตัวของคุณทำให้พายหายไปมาก คุณมีอำนาจสูง ไม่ว่าคุณจะเล็กหรือใหญ่แค่ไหน

### 6. How-to: 3 ขั้นตอนเริ่มเจรจาแบบ Split the Pie ในชีวิตจริง

Nalebuff สรุปวิธีนำหลักการไปใช้จริงใน 3 ขั้นตอน:

**ขั้นตอนที่ 1: ระบุ BATNA ของทุกฝ่าย**

ก่อนเข้าห้องเจรจา ต้องรู้ให้ชัดว่า:
- BATNA ของตัวเองคืออะไร? (ถ้าดีลนี้ไม่สำเร็จ คุณทำอะไรได้บ้าง?)
- BATNA ของอีกฝ่ายน่าจะเป็นอะไร?

**คำถามที่ควรถามตัวเอง:**
- "ถ้าเจรจาล้มเหลวพรุ่งนี้ ฉันจะทำอย่างไร?"
- "อีกฝ่ายมีทางเลือกอื่นอะไรบ้าง?"
- "ใครต้องการดีลนี้มากกว่ากัน?"

**ขั้นตอนที่ 2: คำนวณพายและขยายพาย**

- คำนวณมูลค่าที่จะเกิดขึ้นจากดีล
- ถามตัวเองว่า "เราจะเพิ่มมูลค่าให้ดีลได้อีกอย่างไรบ้าง?"
- มองหาโอกาสที่ทั้งสองฝ่ายได้ประโยชน์เพิ่มขึ้น เช่น การขยายระยะเวลาสัญญา การเพิ่มบริการเสริม หรือการแลกเปลี่ยนทรัพยากรที่แต่ละฝ่ายมีแต่ไม่ได้ใช้

**ตัวอย่างการถามเพื่อขยายพาย:**
> "ถ้าเราขยายดีลนี้ให้ครอบคลุมตลาดเอเชียด้วย มูลค่าจะเพิ่มขึ้นไหม? และถ้าเพิ่ม เราจะแบ่งกันอย่างไรได้บ้าง?"

**ขั้นตอนที่ 3: แบ่งพายอย่างเป็นธรรม**

- เสนอการแบ่ง 50/50 ของ **พาย** (ไม่ใช่มูลค่าดีลทั้งหมด)
- อธิบายหลักการให้อีกฝ่ายเข้าใจ: "BATNA ของเราต่างกัน แต่เราต่างมีส่วนสำคัญในการสร้างพายนี้เท่าๆ กัน"
- ถ้าอีกฝ่ายไม่เห็นด้วย ให้ถามว่า "คุณคิดว่าทำไมคุณควรได้มากกว่าครึ่ง?"

### 7. สรุปข้อคิด: การเจรจาที่ดีที่สุดคือ Positive-Sum Game

Nalebuff ปิดท้ายด้วยหลักการสำคัญ:

**การเจรจาไม่ใช่การต่อสู้ แต่เป็นการร่วมสร้าง**

เมื่อทั้งสองฝ่ายเข้าใจว่าพวกเขากำลังสร้างมูลค่าร่วมกัน ความกลัวและการป้องกันตัวจะลดลง ทำให้ทั้งคู่มุ่งเน้นที่การขยายพายก่อน แล้วค่อยหาวิธีแบ่งที่ยุติธรรม

**3 สัจจะของการเจรจาแบบ Split the Pie:**

1. **BATNA กำหนดจุดตั้งต้น ไม่ใช่ผลลัพธ์** — คุณควรได้อย่างน้อย BATNA ของตัวเอง บวกกับครึ่งหนึ่งของพาย
2. **อำนาจที่แท้จริงคือความขาดไม่ได้** — ยิ่งดีลต้องการคุณมากเท่าไหร่ คุณยิ่งมีอำนาจมากเท่านั้น
3. **ความโปร่งใสสร้างมูลค่า** — การแชร์ข้อมูลอย่างจริงใจมักนำไปสู่ดีลที่ดีกว่าการปิดบัง

## 📚 แนวคิดและคำศัพท์สำคัญ

- **Split the Pie** — หลักการแบ่งมูลค่าที่เกิดขึ้นจากดีลอย่างเท่าเทียม โดยต่างฝ่ายต่างได้ BATNA ของตัวเองก่อน แล้วแบ่งพายที่เหลือ 50/50
- **BATNA (Best Alternative to a Negotiated Agreement)** — ทางเลือกที่ดีที่สุดของคุณถ้าการเจรจาล้มเหลว เป็นตัวกำหนด "พื้น" ของสิ่งที่คุณจะยอมรับ
- **พาย (The Pie)** — มูลค่าเพิ่มที่เกิดขึ้นจากการตกลงร่วมกัน คำนวณโดย: มูลค่าดีล − BATNA ของทุกฝ่าย
- **Zero-Sum Game** — เกมที่ผลรวมเป็นศูนย์ คนหนึ่งได้ต้องมีอีกคนเสีย (การเจรจาแบบเก่า)
- **Positive-Sum Game** — เกมที่ทุกฝ่ายสามารถได้ประโยชน์ร่วมกัน (เป้าหมายของ Split the Pie)
- **ZOPA (Zone of Possible Agreement)** — ช่วงของข้อตกลงที่ทั้งสองฝ่ายยอมรับได้
- **Honest Tea** — แบรนด์ชาออร์แกนิกที่ Barry Nalebuff ร่วมก่อตั้ง ต่อมาขายให้ Coca-Cola ใช้เป็นกรณีศึกษาในหนังสือ
- **Cybersquatter** — ผู้ที่ครอบครองโดเมนเว็บหรือทรัพย์สินทางปัญญาเพื่อขายต่อในราคาสูง
- **Game Theory** — ทฤษฎีเกม คณิตศาสตร์สำหรับวิเคราะห์การตัดสินใจเชิงกลยุทธ์ พื้นฐานของ Split the Pie
- **Anchoring** — การตั้งหมุด กลยุทธ์ที่เสนอตัวเลขแรกเพื่อดึงดูดการเจรจาไปในทิศทางที่ต้องการ

- สถานการณ์ที่คุณเคยเจรจาโดยใช้วิธีเก่า (Zero-Sum) และรู้สึกอึดอัด คืออะไร?
- ถ้าใช้หลัก Split the Pie จะเปลี่ยนผลลัพธ์อย่างไร?
- BATNA ของคุณในการเจรจาครั้งต่อไปคืออะไร?

---

## 📌 บทสรุป

*Split the Pie* ของ Barry Nalebuff พลิกโฉมความเข้าใจการเจรจาด้วยหลักการง่ายแต่ทรงพลัง: **อย่าแย่งชิงพาย แต่ให้ขยายพายก่อน แล้วแบ่งครึ่ง**

แนวคิดนี้ใช้ได้ในทุกบริบท ตั้งแต่การขึ้นเงินเดือน การซื้อขายธุรกิจ ไปจนถึงการตกลงกันในชีวิตประจำวัน กุญแจคือการเปลี่ยนจาก "ฉันจะเอาอะไรได้บ้าง?" เป็น "เราจะสร้างอะไรด้วยกันได้บ้าง?"

การเจรจาที่ดีที่สุดคือการที่ทั้งสองฝ่ายลุกจากโต๊ะไปพร้อมความรู้สึกว่าตัวเองได้รับความยุติธรรม — และนั่นคือผลลัพธ์ที่ Split the Pie ออกแบบมาเพื่อสร้าง
